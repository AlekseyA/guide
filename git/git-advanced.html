# Git, продвинутый уровень

Подборка ресурсов для тех, кто уже успел познакомиться с Гитом и поработать с ним какое-то время:

  • Курс Основы Git на Хекслете. Не официальный курс, более глубокий.
  • Скринкаст Getting Git — видео про внутреннее устройство Гита. Видео местами устарело, но это не уменьшает его ценности.
  • Книга Pro Git — основная книга про Гит, must read. Бесплатная.
  • Руководство Become a git guru от Atlassian (они делают Bitbucket).

Еще, чтобы лучше понять Гит, можно его сделать самому.

# Автоматизация

Автоматизируйте проверку кода (ESLint, Stylelint) при помощи Husky (повесьте хук на пре-комит). Для решения этой задачи нужно познакомиться с NPM.
